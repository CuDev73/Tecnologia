3.12 Caracteres especiales

Como ya se aprendió en las clases anteriores, los archivos HTML son de texto ASCII y no deben contener nada de formato, los únicos caracteres que deben ir son los que se imprimen directamente del teclado. En lenguaje técnico, los caracteres de los archivos HTML deben pertenecer al conjunto estándar de caracteres ASCII (de 7 bits), y no pueden ser ninguno de los caracteres del ASCII "extendido" (de 8 bits).

Para solucionar estos problemas, HTML define una serie especial de códigos, llamados entidades de carácter, que pueden usarse para representar los caracteres que se quiera emplear.

  • Entidades de carácter para caracteres especiales
    • Las entidades e carácter pueden ser de dos formas: entidades nombradas y numeradas
    • Las entidades nombradas empiezan con el signo ampersand (&) y terminan en punto y coma (;). En medio se encuentra el nombre del carácter. Los nombres, distinguen entre mayúsculas y minúsculas, así que tenga cuidado de escribirlos exactamente.
    • Ejemplos: &quot;
    • &laquo;
    • &copy;
    • Las entidades numeradas también empiezan con ampersand y terminan con punto y coma, pero en lugar de nombre llevan el signo de número (#) y un número. Ese número corresponde al conjunto de ISO-Latin-1. Las entidades numeradas son como sigue:
    • Ejemplos: &#130;
    • &#245;
    • En los archivos HTML, las entidades nombradas o numeradas se colocan en el lugar del carácter que representan. Así para poner la palabra francés en el archivo HTML, se escribirá:
    • franc&eacute;s o bien franc&#233;s
  • Entidades de carácter para caracteres reservados
    • Existen unos cuantos caracteres que tienen un significado especial en el propio HTML. Para estas excepciones también se deben usar las entidades de carácter. Por ejemplo: se quiere poner una línea de código como la siguiente:
    • <P><CODE>if x < 0 do print i</CODE></P>
    • No parece nada raro, sin embargo, el problema es con el signo (<) porque para HTML significa "el inicio de una etiqueta" y también con el signo (>) que significa "el final de la etiqueta". La escritura correcta en HTML de la línea de código anterior, sería como sigue:
    • <P><CODE>if x &lt; 0 do print i</CODE></P>
    • HTML cuenta con caracteres reservados, para cada uno de estos caracteres, según se muestra:

Entidad Resultado

  • &lt; <
  • &gt; >
  • &amp; &
  • &quot; "

Lista de códigos o caracteres especiales

  • &aacute; á &Aacute; Á &ntilde; ñ &#191; ¿
  • &eacute; é &Eacute; É &Ntilde; Ñ &#161; ¡
  • &iacute; í &Iacute; Í &uuml; ü
  • &oacute; ó &Oacute; Ó &Uuml; Ü
  • &uacute; ú &Uacute; Ú

3.13 Formateo del <BODY> - Atributos

Se ha explicado que en el "cuerpo" del documento <BODY>, va todo lo queramos mostrar en nuestra página web. Esta etiqueta tiene atributos, con los que se pueden personalizar según el aspecto que se desee. Esta personalización se consigue a través de los siguientes atributos:

  • Bgcolor: Color de fondo
    • Con este atributo podremos definir el color de fondo para nuestra página.
    • Ejemplo:
    • Si queremos que nuestra página tenga un fondo rojo
    • <body bgcolor="red"></body>........................color=nombre en ingles
    • O bien <body bgcolor="#FF0000"></body>.................color=hexadecimal
  • Text: Color de texto
    • Se usa para predefinir el color del texto de nuestra web.
    • Ejemplo:
    • Si queremos que nuestro texto sea en negro:
    • <body text= "#000000"></body>
  • Link, alink y vlink: Color de enlaces
    • Para definir el color con el que se mostraran los links de nuestra página, debemos diferenciar tres tipos de instrucciones: el color del enlace (con la etiqueta "link"), el color del link activo (con la etiqueta "alink") y el color del link ya visitado (con la etiqueta "vlink").
    • Ejemplo:
    • Si queremos que nuestra página tenga un color de enlace rojo y que cuando estén activos también sean rojos y que cuando estén inactivos se queden en azul, deberemos escribir lo siguiente:
    • <body link= "#FF0000" alink= "#FF0000" vlink= "#0000FF"></body>
  • Background: Imagen de fondo
    • Se usa para que tu página tenga una imagen de fondo.
    • Ejemplo:
    • <body background="nombre de la imagen o URL"> </body>
    • Obs.: si la imagen no es suficientemente grande para rellenar toda la página, la imagen se repetirá a lo ancho y a lo largo hasta rellenar todo el espacio.
  • Background = "" bgproperties = "fixed": Imagen de fondo fija
    • Se usa cuando se especifica una imagen como fondo, se añade este atributo para indicar si la imagen es fija o desplazable.
    • Ejemplo:
    • <body background = "back.gif" bgproperties = "fixed">
  • Topmargin, bottommargin, leftmargin, rightmargin: Márgenes
    • Se usa para predefinir los márgenes de nuestra página web, "leftmargin" para el margen izquierdo, "topmargin" para el margen de arriba, "rightmargin" para el margen de la derecha y "bottommargin" para el margen de abajo.
    • Ejemplo:
    • Si queremos que nuestros márgenes sean de 10 píxeles por todas partes:
    • <body leftmargin="10px" topmargin="10px" rightmargin="10px" bottommargin="10px"></body>

3.14 Tamaño, color y tipos de fuentes

La etiqueta <FONT> </FONT> se usa para controlar las características de una cadena de caracteres, que no está cubierta por los estilos de caracteres inicialmente definidos en el lenguaje HTML. Se usa para cambiar el tipo de letra, el tamaño, y el color.

Ejemplo:

Si queremos cambiar el tipo de letra

<font face = "Times New Roman"> fuente de la fuente </font>

Si queremos cambiar el tamaño

<font size="5">Este es el tamaño de fuente 5</font>

Si queremos cambiar el color

<font color = "#FF0000"> texto rojo </font>
