4.1 Listas

Uno de los elementos más comunes en HTML, son las listas. Se definen cinco clases de listas, pero las más usadas son: Listas Ordenadas o Numeradas

  • Listas Desordenadas o con Viñetas
  • Listas de Glosario o Definición
  • Comandos o Etiquetas de Listas
    • Todas las listas están encerradas por las etiquetas de apertura y cierre correspondiente a su tipo (<UL> </UL>, <OL> </OL>, etc.); y cada elemento de la lista tiene su propia etiqueta: <DT> y <DD> en listas de glosario de términos y <LI> en las demás.
    • Para escribir los códigos de listas, se sugiere tener un cierto orden: en una línea escribir los comandos de listas, y cada elemento de las listas en otra línea y con una identación. De esta manera se podrá distinguir fácilmente todas las listas y cada uno de sus elementos.

4.2 Listas Ordenadas o Numeradas

Las listas ordenadas son listas donde cada elemento está numerado. Van encerradas en las etiquetas <OL> </OL>, y cada elemento de la lista empieza con la etiqueta <LI>, que va de un solo lado, no es obligatorio poner el cierre.

Ejemplo:

<OL>

  • <LI> María
  • <LI> Miguel
  • <LI> Juan

</OL>

Los atributos son partes adicionales de las etiquetas HTML, que contienen opciones u otra información acerca de la etiqueta en sí.

  • Atributos
    • type Para elegir el tipo de numeración
    • start Inicio de numeración
  • Tipos de numeración
    • "1" Números ordinales (Ej: 1, 2, 3, 4...), es la numeración predeterminada
    • "I" Números romanos en mayúsculas (Ej: I, II, III...)
    • "i" Números romanos en minúsculas (Ej: i, ii, iii, ...)
    • "a" Letras en minúsculas (Ej. a, b, c....)
    • "A" Letras en mayúsculas (Ej. A, B, C....)

Sintaxis:

<Etiqueta Atributo = " " > ... ... ... ... ... ... .... </Etiqueta>

Ejemplo:

<OL Type = "I">

  • <LI> ... ... ... ... ... ....

</OL>

<OL type = "1" start = "4">

  • <LI> ... ... ... ... ... ....

</OL>

4.3 Listas Desordenadas o con Viñetas

En las listas con viñetas los elementos están marcados con una viñeta; es decir, un símbolo tipográfico especial. Estas listas se ven igual que las numeradas, excepto que las listas se indican con las etiquetas <UL> </UL>, y los elementos de la lista también se indican con la etiqueta <LI>.

Ejemplo:

<UL>

  • <LI> María
  • <LI> Miguel
  • <LI> Juan

</UL>

  • Atributos
    • type Para elegir el tipo de viñetas
  • Tipos de viñetas
    • "disc" Viñetas redondas rellenadas, es la viñeta predeterminada
    • "square" Viñetas cuadradas
    • "circle" Viñetas circulares sin relleno

Ejemplo:

<UL type = "square">

  • <LI> ... ... ... ... ... ....

</UL>

4.4 Listas de glosario o definición

En las listas de glosario, cada elemento tiene dos partes: el término y su definición. Por ello, en ocasiones a estas listas se les llama también de definición. Estas dos partes tienen sus propias etiquetas: <DT> para el término, y <DD> para su definición; van de un solo lado, y se usan en pares. Todas las listas de glosario están indicadas con las etiquetas <DL> </DL>.

Ejemplo:

<DL>

  • <DT> HTML
  • <DD> Lenguaje de marcado de hipertexto.
  • <DT> WWW:
  • <DD> Es un sistema gráfico de información de hipertexto, distribuido, global, interactivo, dinámico e independiente de la plataforma, que funciona en Internet.

</DL>

4.5 Listas Anidadas

El lenguaje HTML nos permite insertar una lista dentro de otra. Se pueden anidar listas de distinto tipo, por ejemplo, podemos tener una lista desordenada y uno de los ítems puede ser una lista ordenada.

Para el anidamiento de listas solo debemos tener cuidado en la correcta apertura y cerrado de las mismas.

Ejemplo:

<UL>

  • <LI>Elemento 1</LI>
  • <LI>Elemento 2</LI>

<OL>

  • <LI>Elemento 2.1</LI>
  • <LI>Elemento 2.2</LI>
  • ...
  • <LI>Elemento 2.N</LI>

</OL>

  • <LI>Elemento 3</LI>
  • ...
  • <LI>Elemento N</LI>

</UL>
